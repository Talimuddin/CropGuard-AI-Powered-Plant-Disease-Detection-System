# 🌱 CropGuard – AI Powered Plant Disease Detection System

CropGuard is an AI-powered web application that helps farmers identify plant diseases from crop images. The system uses Gemini AI to analyze uploaded images and provides disease diagnosis, treatment recommendations, medicine suggestions, and prevention tips.


## 🚀 Features
- Plant Disease Detection from Images
- AI-Powered Disease Diagnosis
- Treatment Recommendations
- Medicine Suggestions
- Prevention Tips
- Multilingual Support
- Text-to-Speech (Voice Assistance)
- Responsive and User-Friendly Interface


## 🛠️ Technologies Used
- Python
- Flask
- Gemini AI API
- HTML
- CSS
- JavaScript

## 📸 How It Works
1. Upload a crop image.
2. CropGuard analyzes the image using Gemini AI.
3. The system identifies possible diseases.
4. Users receive:
    - Disease Name
    - Symptoms
    - Treatment Suggestions
    - Medicine Recommendations
    - Prevention Tips
